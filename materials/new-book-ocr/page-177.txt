RESPONSE TO COMPLEX INPUTS
The LTIC system response discussed so far applies to general input signals, real or complex. However, if the system is real, that is, if h(t) is real, then we shall show that the real part of the input generates the real part of the output, and a similar conclusion applies to the imaginary part.
If the input is x(t) = x_r(t) + jx_i(t), where x_r(t) and x_i(t) are the real and imaginary parts of x(t), then for real h(t)

y(t) = h(t) * [x_r(t) + jx_i(t)] = h(t) * x_r(t) + jh(t) * x_i(t) = y_r(t) + jy_i(t)

where y_r(t) and y_i(t) are the real and the imaginary parts of y(t). Using the right-directed-arrow notation to indicate a pair of the input and the corresponding output, the foregoing result can be expressed as follows. If

x(t) = x_r(t) + jx_i(t)  \Longrightarrow  y(t) = y_r(t) + jy_i(t)

then

x_r(t)  \Longrightarrow  y_r(t)    and    x_i(t)  \Longrightarrow  y_i(t)                                                   (2.31)
