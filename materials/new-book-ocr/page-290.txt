Figure 3.22f shows the general situation for n ≥ 0. The two functions x[m] and g[n − m] overlap over the interval 0 ≤ m ≤ n. Therefore,

c[n] = ∑_{m=0}^{n} x[m]g[n − m]

= ∑_{m=0}^{n} (0.8)^m(0.3)^{n−m}

= (0.3)^n ∑_{m=0}^{n} \left(\frac{0.8}{0.3}\right)^m

= 2[(0.8)^{n+1} − (0.3)^{n+1}]    n ≥ 0    (see Sec. B.8-3)

For n < 0, there is no overlap between x[m] and g[n − m], as shown in Fig. 3.22g, so that

c[n] = 0    n < 0

Combining pieces, we see that

c[n] = 2[(0.8)^{n+1} − (0.3)^{n+1}]u[n]

which agrees with the result found earlier in Ex. 3.20.
