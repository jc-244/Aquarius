3.8 SYSTEM RESPONSE TO EXTERNAL INPUT:
THE ZERO-STATE RESPONSE

The zero-state response y[n] is the system response to an input x[n] when the system is in the
zero state. In this section we shall assume that systems are in the zero state unless mentioned
otherwise, so that the zero-state response will be the total response of the system. Here we follow
the procedure parallel to that used in the continuous-time case by expressing an arbitrary input
x[n] as a sum of impulse components. A signal x[n] in Fig. 3.20a can be expressed as a sum
of impulse components, such as those depicted in Figs. 3.20bŌĆō3.20f. The component of x[n] at
n = m is x[m]╬┤[n ŌłÆ m], and x[n] is the sum of all these components summed from m = ŌłÆŌł× to Ōł×.

For a linear system, if we know the system response to impulse ╬┤[n], we can obtain the system
response to any arbitrary input by summing the system response to various impulse components.
Let h[n] be the system response to impulse input ╬┤[n]. We shall use the notation

x[n] Ō¤╣ y[n]

to indicate the input and the corresponding response of the system. Thus, if

╬┤[n] Ō¤╣ h[n]

then because of time invariance

╬┤[n - m] Ō¤╣ h[n - m]

and because of linearity

x[m]╬┤[n - m] Ō¤╣ x[m]h[n - m]

and again because of linearity

Ōłæ_{m=-Ōł×}^{Ōł×} x[m]╬┤[n - m]  Ō¤╣  Ōłæ_{m=-Ōł×}^{Ōł×} x[m]h[n - m]
        ŌÅ¤x[n]                         ŌÅ¤y[n]

The left-hand side is x[n] [see Eq. (3.30)], and the right-hand side is the system response y[n] to
input x[n]. Therefore,ŌĆĀ

y[n] = Ōłæ_{m=-Ōł×}^{Ōł×} x[m]h[n - m]                                              (3.31)

The summation on the right-hand side is known as the convolution sum of x[n] and h[n], and is
represented symbolically by x[n] ŌłŚ h[n]

x[n] ŌłŚ h[n] = Ōłæ_{m=-Ōł×}^{Ōł×} x[m]h[n - m]

PROPERTIES OF THE CONVOLUTION SUM
The structure of the convolution sum is similar to that of the convolution integral. Moreover,
the properties of the convolution sum are similar to those of the convolution integral. We shall
enumerate these properties here without proof. The proofs are similar to those for the convolution
integral and may be derived by the reader.

ŌĆĀ In deriving this result, we have assumed a time-invariant system. The system response to input ╬┤[n - m]
for a time-varying system cannot be expressed as h[n - m]; instead, it has the form h[n, m]. Using this form,
Eq. (3.31) is modified as follows:

y[n] = Ōłæ_{m=-Ōł×}^{Ōł×} x[m]h[n, m]

The Commutative Property.

x1[n] ŌłŚ x2[n] = x2[n] ŌłŚ x1[n]

The Distributive Property.

x1[n] ŌłŚ (x2[n] + x3[n]) = x1[n] ŌłŚ x2[n] + x1[n] ŌłŚ x3[n]

The Associative Property.

x1[n] ŌłŚ (x2[n] ŌłŚ x3[n]) = (x1[n] ŌłŚ x2[n]) ŌłŚ x3[n]

The Shifting Property.  If

x1[n] ŌłŚ x2[n] = c[n]

then

x1[n ŌłÆ m] ŌłŚ x2[n ŌłÆ p] = c[n ŌłÆ m ŌłÆ p]                                                      (3.32)

The Convolution with an Impulse.

x[n] ŌłŚ ╬┤[n] = x[n]

The Width Property. If x1[n] and x2[n] have finite widths of W1 and W2, respectively, then the
width of x1[n] ŌłŚ x2[n] is W1 + W2. The width of a signal is 1 less than the number of its elements
(length). Thus the signal in Fig. 3.22h has six elements (length of 6) but a width of only 5.
Alternately, the property may be stated in terms of lengths as follows: if x1[n] and x2[n] have
finite lengths of L1 and L2 elements, respectively, then the length of x1[n] ŌłŚ x2[n] is L1 + L2 ŌłÆ 1
elements.

CAUSALITY AND ZERO-STATE RESPONSE
In deriving Eq. (3.31), we assumed the system to be linear and time-invariant. There were no other
restrictions on either the input signal or the system. In our applications, almost all the input signals
are causal, and a majority of the systems are also causal. These restrictions further simplify the
limits of the sum in Eq. (3.31). If the input x[n] is causal, x[m] = 0 for m < 0. Similarly, if the
system is causal (i.e., if h[n] is causal), then h[x] = 0 for negative x so that h[n ŌłÆ m] = 0 when
m > n. Therefore, if x[n] and h[n] are both causal, the product x[m]h[n ŌłÆ m] = 0 for m < 0 and for
m > n, and it is nonzero only for the range 0 Ōēż m Ōēż n. Therefore, Eq. (3.31) in this case reduces to

y[n] = Ōłæ_{m=0}^{n} x[m]h[n ŌłÆ m]                                                           (3.33)

We shall evaluate the convolution sum first by an analytical method and later with graphical
aid.

EXAMPLE 3.20  Convolution of Causal Signals

Determine c[n] = x[n] * g[n] for

x[n] = (0.8)^n u[n]    and    g[n] = (0.3)^n u[n]

We have

c[n] = \sum_{m=-\infty}^{\infty} x[m]g[n - m]

Note that

x[m] = (0.8)^m u[m]    and    g[n - m] = (0.3)^{n-m} u[n - m]

Both x[n] and g[n] are causal. Therefore [see Eq. (3.33)],

c[n] = \sum_{m=0}^{n} x[m]g[n - m] = \sum_{m=0}^{n} (0.8)^m u[m](0.3)^{n-m} u[n - m]

In this summation, m lies between 0 and n (0 \le m \le n). Therefore, if n \ge 0, then both m and
n - m \ge 0 so that u[m] = u[n - m] = 1. If n < 0, m is negative because m lies between 0 and n,
and u[m] = 0. Therefore,

c[n] = {
\sum_{m=0}^{n} (0.8)^m (0.3)^{n-m}    n \ge 0
0                                           n < 0

or

c[n] = (0.3)^n \sum_{m=0}^{n} \left( \frac{0.8}{0.3} \right)^m u[n]

This is a geometric progression with common ratio (0.8/0.3). From Sec. B.8-3 we have

c[n] = (0.3)^n \frac{(0.8)^{n+1} - (0.3)^{n+1}}{(0.3)^n (0.8 - 0.3)} u[n]

= 2[(0.8)^{n+1} - (0.3)^{n+1}]u[n]

RESPONSE TO COMPLEX INPUTS
As in the case of real continuous-time systems, we can show that for an LTID system with real h[n], if the input and the output are expressed in terms of their real and imaginary parts, then the real part of the input generates the real part of the response and the imaginary part of the input generates the imaginary part. Thus, if

x[n] = x_r[n] + jx_i[n]      and      y[n] = y_r[n] + jy_i[n]

using the right-directed arrow to indicate the inputŌĆōoutput pair, we can show that

x_r[n] Ō¤╣ y_r[n]      and      x_i[n] Ō¤╣ y_i[n]                                           (3.34)

The proof is similar to that used to derive Eq. (2.31) for LTIC systems.

MULTIPLE INPUTS
Multiple inputs to LTI systems can be treated by applying the superposition principle. Each input is considered separately, with all other inputs assumed to be zero. The sum of all these individual system responses constitutes the total system output when all the inputs are applied simultaneously.
