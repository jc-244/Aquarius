3.13 SUMMARY
This chapter discusses time-domain analysis of LTID (linear, time-invariant, discrete-time) systems. The analysis is parallel to that of LTIC systems, with some minor differences. Discrete-time systems are described by difference equations. For an Nth-order system, N auxiliary conditions must be specified for a unique solution. Characteristic modes are discrete-time exponentials of the form γ^n corresponding to an unrepeated root γ, and the modes are of the form n^iγ^n corresponding to a repeated root γ.

The unit impulse function δ[n] is a sequence of a single number of unit value at n = 0. The unit impulse response h[n] of a discrete-time system is a linear combination of its characteristic modes.‡

The zero-state response (response due to external input) of a linear system is obtained by breaking the input into impulse components and then adding the system responses to all the impulse components. The sum of the system responses to the impulse components is in the form of a sum, known as the convolution sum, whose structure and properties are similar to the convolution integral. The system response is obtained as the convolution sum of the input x[n] with the system’s impulse response h[n]. Therefore, the knowledge of the system’s impulse response allows us to determine the system response to any arbitrary input.
