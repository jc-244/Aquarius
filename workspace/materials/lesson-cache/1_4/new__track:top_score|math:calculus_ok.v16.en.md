## 📖 第2页：单位阶跃函数 $$u(t)$$

---

### 1.4-1 单位阶跃函数 $$u(t)$$

---

### 🔑 定义

单位阶跃函数的数学定义非常简单：

$$\boxed{u(t) = \begin{cases} 1 & t \geq 0 \\ 0 & t < 0 \end{cases}}$$

用大白话说：**在 $$t=0$$ 这一刻"跳起来"，之前是0，之后是1。**

---

### 📊 图形理解

```
u(t)
  1 |————————————————→
    |
  0 |____
    ————————————————→ t
         0
```

- $$t < 0$$ 时，函数值为 **0**（信号还没开始）
- $$t \geq 0$$ 时，函数值为 **1**（信号开始了）

---

### 🔧 核心用途一：让信号"从零时刻开始"

如果我们有一个从 $$t = -\infty$$ 就存在的永恒指数信号 $$e^{-at}$$，想让它**从 $$t=0$$ 才开始**，只需乘以 $$u(t)$$：

$$e^{-at} \cdot u(t) = \begin{cases} e^{-at} & t \geq 0 \\ 0 & t < 0 \end{cases}$$

> 🎯 **口诀**：乘以 $$u(t)$$，信号从0开始；乘以 $$u(t-T)$$，信号从 $$T$$ 开始。

---

### 🔧 核心用途二：描述矩形脉冲（门函数）

**矩形脉冲**可以用两个阶跃函数相减来表示：

$$x(t) = u(t-2) - u(t-4)$$

这表示：
- $$u(t-2)$$：在 $$t=2$$ 时"跳起来"
- $$-u(t-4)$$：在 $$t=4$$ 时"跳回去"
- 合起来：在 $$[2, 4]$$ 区间内值为1，其余为0

```
x(t)
  1 |      |‾‾‾‾‾‾|
    |      |      |
  0 |______|      |______→ t
         2       4
```

---

### 🔧 核心用途三：描述分段函数

**例题1.6** 用阶跃函数描述三角形信号（图1.16a）

**思路**：把复杂的分段信号拆成几个简单部分之和。

**第一段** $$x_1(t)$$（斜率为1的斜坡，在 $$[0,2]$$ 内）：

$$x_1(t) = t \cdot [u(t) - u(t-2)]$$

**第二段** $$x_2(t)$$（斜率为-2的斜坡，在 $$[2,3]$$ 内，在 $$t=3$$ 处为零）：

斜坡方程：$$-2(t-3)$$（在 $$t=3$$ 时值为0，斜率为-2 ✓）

$$x_2(t) = -2(t-3) \cdot [u(t-2) - u(t-3)]$$

**合并**：

$$\boxed{x(t) = t\,u(t) - 3(t-2)\,u(t-2) + 2(t-3)\,u(t-3)}$$

---

### 📝 延时阶跃函数 $$u(t-T)$$

$$u(t-T) = \begin{cases} 1 & t \geq T \\ 0 & t < T \end{cases}$$

把 $$u(t)$$ 向右平移 $$T$$ 秒，就得到 $$u(t-T)$$。

---

### ✅ 本页小结

| 操作 | 表达式 | 含义 |
|------|--------|------|
| 信号从0开始 | $$x(t) \cdot u(t)$$ | 0之前清零 |
| 信号从T开始 | $$x(t) \cdot u(t-T)$$ | T之前清零 |
| 矩形门函数 | $$u(t-a) - u(t-b)$$ | 在$$[a,b]$$内为1 |

## 📖 第3页：单位冲激函数 $$\delta(t)$$ — 定义与直觉

---

### 1.4-2 单位冲激函数 $$\delta(t)$$（上）

---

### 🔑 Dirac 的经典定义

单位冲激函数由物理学家 **P. A. M. Dirac** 定义，分两部分：

$$\boxed{\delta(t) = 0 \quad (t \neq 0), \qquad \int_{-\infty}^{+\infty} \delta(t)\,dt = 1}$$

用大白话说：
- **除了 $$t=0$$ 这一点，冲激函数处处为零**
- **但它在 $$t=0$$ 处的"面积"（积分）等于1**

---

### 🎨 直觉理解：极限矩形脉冲

想象一个矩形脉冲：
- 宽度为 $$\varepsilon$$（非常小）
- 高度为 $$\frac{1}{\varepsilon}$$（非常大）
- 面积 = 宽 × 高 = $$\varepsilon \times \frac{1}{\varepsilon} = 1$$（始终保持为1）

当 $$\varepsilon \to 0$$ 时：
- 宽度 → 0（无限窄）
- 高度 → ∞（无限高）
- 面积 **始终 = 1**

这个极限就是 $$\delta(t)$$！

```
高度
1/ε |  |‾‾|
    |  |  |
    |  |  |
  0 |__|  |__→ t
      -ε/2 ε/2
      
当ε→0，变成一根"无限高的针"
```

---

### 📌 图形表示

由于冲激函数无法真正画出来，我们用一个**带箭头的竖线**表示，旁边标注强度：

```
δ(t)
  ↑ (1)
  |
——+——————→ t
  0
```

括号里的数字表示冲激的**强度（面积）**。

---

### 🔄 其他形式的冲激近似

不只是矩形脉冲，以下函数在极限情况下也趋近于 $$\delta(t)$$：

**指数脉冲**：$$\alpha e^{-\alpha t} u(t)$$，当 $$\alpha \to \infty$$

验证：
$$\int_0^{\infty} \alpha e^{-\alpha t}\,dt = \alpha \cdot \frac{1}{\alpha} = 1 \quad \checkmark$$

**关键思想**：冲激函数的本质不在于形状，而在于：
> **有效持续时间趋近于零，同时面积保持为1。**

---

### ⚡ 强度为 $$k$$ 的冲激函数

$$k\delta(t)$$：在 $$t=0$$ 处的冲激，面积（强度）为 $$k$$

$$\int_{-\infty}^{+\infty} k\delta(t)\,dt = k$$

---

### ✅ 本页小结

| 性质 | 描述 |
|------|------|
| 位置 | 只在 $$t=0$$ 处"存在" |
| 其余位置 | $$\delta(t) = 0$$ |
| 面积 | $$\int_{-\infty}^{+\infty}\delta(t)\,dt = 1$$ |
| 图形 | 带箭头的竖线，旁标强度 |

---

## 📖 第4页：单位冲激函数 $$\delta(t)$$ — 重要性质

---

### 1.4-2 单位冲激函数 $$\delta(t)$$（下）

---

### 🌟 性质一：与函数相乘

设 $$\varphi(t)$$ 在 $$t=0$$ 处连续，则：

$$\boxed{\varphi(t)\,\delta(t) = \varphi(0)\,\delta(t)}$$

**理解**：冲激函数只在 $$t=0$$ 处"有效"，所以 $$\varphi(t)$$ 乘以 $$\delta(t)$$ 时，$$\varphi(t)$$ 只有在 $$t=0$$ 处的值 $$\varphi(0)$$ 起作用。

**推广**（冲激在 $$t=T$$ 处）：

$$\boxed{\varphi(t)\,\delta(t-T) = \varphi(T)\,\delta(t-T)}$$

**例子**：
$$e^{-2t}\,\delta(t) = e^{0}\,\delta(t) = \delta(t)$$

$$(t^3+3)\,\delta(t) = (0+3)\,\delta(t) = 3\,\delta(t)$$

---

### 🌟 性质二：采样性质（最重要！）

由上面的乘法性质，两边积分：

$$\boxed{\int_{-\infty}^{+\infty} \varphi(t)\,\delta(t-T)\,dt = \varphi(T)}$$

**这就是冲激函数的采样性质（或筛选性质）！**

**含义**：把函数 $$\varphi(t)$$ 与位于 $$t=T$$ 的冲激相乘再积分，结果就是 $$\varphi(t)$$ 在 $$t=T$$ 处的值。

> 🎯 **口诀**：冲激在哪里，就"采样"哪里的值！

**例子**：

$$\int_{-\infty}^{+\infty} e^{-j\omega t}\,\delta(t)\,dt = e^{0} = 1$$

$$\int_{-\infty}^{+\infty} \cos\!\left(\frac{\pi t}{4}\right)\delta(t-2)\,dt = \cos\!\left(\frac{\pi \cdot 2}{4}\right) = \cos\!\left(\frac{\pi}{2}\right) = 0$$

---

### 🌟 性质三：冲激函数是阶跃函数的导数

这是一个非常优美的结论：

$$\boxed{\frac{d\,u(t)}{dt} = \delta(t)}$$

**证明思路**（广义函数方法）：

对 $$\int_{-\infty}^{+\infty} \frac{du}{dt}\,\varphi(t)\,dt$$ 用分部积分：

$$= u(t)\varphi(t)\Big|_{-\infty}^{+\infty} - \int_{-\infty}^{+\infty} u(t)\dot{\varphi}(t)\,dt$$

$$= \varphi(\infty) - \int_0^{\infty} \dot{\varphi}(t)\,dt = \varphi(\infty) - [\varphi(\infty) - \varphi(0)] = \varphi(0)$$

这正好满足采样性质！所以 $$\frac{du}{dt}$$ 就是 $$\delta(t)$$ ✓

**反过来**：

$$\boxed{\int_{-\infty}^{t} \delta(\tau)\,d\tau = u(t)}$$

---

### 📊 奇异函数家族

从冲激函数出发，通过积分和求导，可以得到一系列**奇异函数**：

```
... ← 求导 ← 求导 ← 求导
冲激的导数  →  δ(t)  →  u(t)  →  tu(t)  →  t²/2·u(t)  →  ...
（广义函数）    冲激    阶跃     斜坡       抛物线
                ← 积分 ← 积分 ← 积分 ← 积分
```

---

### ✅ 本页小结

| 性质名称 | 公式 |
|---------|------|
| 乘法性质 | $$\varphi(t)\delta(t-T) = \varphi(T)\delta(t-T)$$ |
| 采样性质 | $$\int_{-\infty}^{+\infty}\varphi(t)\delta(t-T)\,dt = \varphi(T)$$ |
| 导数关系 | $$\frac{du(t)}{dt} = \delta(t)$$ |
| 积分关系 | $$\int_{-\infty}^{t}\delta(\tau)\,d\tau = u(t)$$ |

## 📖 第5页：指数函数 $$e^{st}$$ 与复频率

---

### 1.4-3 指数函数 $$e^{st}$$

---

### 🔑 复数指数的定义

$$e^{st}$$ 中，$$s$$ 是**复数**：

$$s = \sigma + j\omega$$

展开 $$e^{st}$$：

$$\boxed{e^{st} = e^{(\sigma+j\omega)t} = e^{\sigma t}e^{j\omega t} = e^{\sigma t}(\cos\omega t + j\sin\omega t)}$$

其中：
- $$\sigma$$：**奈培频率（实部）**，控制幅度的增减速率
- $$\omega$$：**角频率（虚部）**，控制振荡的快慢
- $$e^{\sigma t}$$：幅度包络
- $$e^{j\omega t}$$：振荡部分（欧拉公式）

---

### 🎭 $$e^{st}$$ 的"变身术"

$$e^{st}$$ 是一个**万能函数**，通过改变 $$s$$ 的值，可以变成各种常见信号：

| 情况 | $$s$$ 的值 | 对应信号 | 描述 |
|------|-----------|---------|------|
| $$\sigma=0, \omega=0$$ | $$s=0$$ | 常数 $$k$$ | 直流信号 |
| $$\omega=0$$ | $$s=\sigma$$ | $$e^{\sigma t}$$ | 单调指数 |
| $$\sigma=0$$ | $$s=\pm j\omega$$ | $$\cos\omega t$$ | 等幅正弦 |
| $$\sigma\neq0,
