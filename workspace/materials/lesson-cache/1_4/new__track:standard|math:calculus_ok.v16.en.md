## 📖 第2页：单位阶跃函数 u(t)

### 1.4-1 单位阶跃函数 u(t)

---

#### 🔵 定义

单位阶跃函数 \(u(t)\) 的数学定义为：

$$\boxed{u(t) = \begin{cases} 1 & t \geq 0 \\ 0 & t < 0 \end{cases}}$$

用大白话说：**\(t=0\) 之前它是0，\(t=0\) 及之后它是1**。就像一个开关，在 \(t=0\) 时刻"啪"地打开了！

---

#### 📊 图形理解

```
u(t)
  1 |————————————————→
    |
  0 |←————
    +——————————————→ t
         0
```

---

#### 🔧 核心用途一：让信号"从零开始"

如果我们有一个从 \(t = -\infty\) 就存在的永恒指数信号 \(e^{-at}\)，想让它**从 \(t=0\) 才开始**，只需乘以 \(u(t)\)：

$$e^{-at} \cdot u(t) = \begin{cases} e^{-at} & t \geq 0 \\ 0 & t < 0 \end{cases}$$

> 🌟 **\(u(t)\) 就像一个"时间开关"**，乘上它就能让任何信号变成因果信号（causal signal）。

---

#### 🔧 核心用途二：构造矩形脉冲（门函数）

**延迟的阶跃函数**：\(u(t-T)\) 表示在 \(t=T\) 时刻才"打开"的阶跃。

$$u(t-T) = \begin{cases} 1 & t \geq T \\ 0 & t < T \end{cases}$$

**两个阶跃相减 = 矩形脉冲（门函数）**：

$$x(t) = u(t-2) - u(t-4)$$

```
x(t)
  1 |     |‾‾‾‾‾‾‾|
    |     |       |
  0 |—————+———————+———→ t
         2       4
```

这个信号在 \([2, 4)\) 上等于1，其余为0。

---

#### 🔧 核心用途三：表示分段信号

**例题1.6 思路**：把一个三角形信号分解为两部分

对于图1.16a中的三角信号，分解步骤：

**第一段** \([0, 2)\)：斜率为1的斜坡，乘以门函数：
$$x_1(t) = t \cdot [u(t) - u(t-2)]$$

**第二段** \([2, 3)\)：斜率为 \(-2\)，零点在 \(t=3\)，即 \(-2(t-3)\)，乘以门函数：
$$x_2(t) = -2(t-3)[u(t-2) - u(t-3)]$$

**合并后化简**：

$$\boxed{x(t) = t\,u(t) - 3(t-2)\,u(t-2) + 2(t-3)\,u(t-3)}$$

> 💡 **技巧**：每当信号的数学描述在某个时刻 \(T\) 发生变化，就在那里加入一个 \(u(t-T)\) 项。

---

#### 📝 小结

| 操作 | 表达式 | 含义 |
|------|--------|------|
| 因果化 | \(f(t)\cdot u(t)\) | \(t<0\) 时为零 |
| 延迟开关 | \(u(t-T)\) | \(t=T\) 时打开 |
| 门函数 | \(u(t-a)-u(t-b)\) | 仅在 \([a,b)\) 为1 |

## 📖 第3页：单位冲激函数 δ(t) —— 定义与性质

### 1.4-2 单位冲激函数 δ(t)

---

#### 🔵 Dirac 经典定义

单位冲激函数由两个条件共同定义：

$$\boxed{\delta(t) = 0 \quad (t \neq 0), \qquad \int_{-\infty}^{+\infty} \delta(t)\,dt = 1}$$

用大白话说：
- **除了 \(t=0\) 之外，它处处为零**
- **但它在 \(t=0\) 处的"面积"（积分）等于1**

这意味着它在 \(t=0\) 处无限高、无限窄，却保持面积为1。

---

#### 📊 直观理解：极限矩形脉冲

想象一个宽度为 \(\varepsilon\)、高度为 \(\frac{1}{\varepsilon}\) 的矩形脉冲：

```
高度 1/ε
    |  |‾‾‾|  |
    |  |   |  |      面积 = ε × (1/ε) = 1 ✓
————+——+———+——+————→ t
      -ε/2  ε/2
```

当 \(\varepsilon \to 0\) 时：高度 \(\to \infty\)，宽度 \(\to 0\)，面积始终 \(= 1\)。

> 🎯 **关键**：冲激函数的本质不在于形状，而在于**面积为1、持续时间趋于零**。

---

#### 🔵 与函数相乘的性质

设 \(\varphi(t)\) 在 \(t=0\) 处连续，则：

$$\boxed{\varphi(t)\,\delta(t) = \varphi(0)\,\delta(t)}$$

**推广到 \(t=T\) 处的冲激**（\(\varphi(t)\) 在 \(t=T\) 连续）：

$$\boxed{\varphi(t)\,\delta(t-T) = \varphi(T)\,\delta(t-T)}$$

**直觉解释**：冲激只在 \(t=T\) 处"有效"，所以 \(\varphi(t)\) 在其他地方的值都无关紧要，只有 \(\varphi(T)\) 这一个值被"抓住"了。

---

#### 🔵 采样（筛选）性质 ⭐⭐⭐

这是冲激函数**最重要**的性质！

$$\boxed{\int_{-\infty}^{+\infty} \varphi(t)\,\delta(t-T)\,dt = \varphi(T)}$$

**含义**：把函数 \(\varphi(t)\) 与位于 \(t=T\) 的冲激相乘再积分，结果就是 \(\varphi(t)\) 在 \(t=T\) 处的值。

> 🌟 **冲激函数就像一个"采样器"**——它能从连续信号中"抓取"某一时刻的值！

**验证示例**（Drill 1.10a）：

$$\int_{-\infty}^{+\infty} \delta(t)\,e^{-j\omega t}\,dt = e^{-j\omega \cdot 0} = 1 \checkmark$$

---

#### 🔵 广义函数视角

经典定义有数学漏洞（\(t=0\) 处未定义），更严格的方式是用**采样性质来定义冲激**：

> 单位冲激 \(\delta(t-T)\) 是这样一个"函数"：对任意在 \(t=T\) 连续的 \(\varphi(t)\)，都有
> $$\int_{-\infty}^{+\infty} \varphi(t)\,\delta(t-T)\,dt = \varphi(T)$$

---

#### 🔵 冲激与阶跃的关系 ⭐

$$\boxed{\frac{d\,u(t)}{dt} = \delta(t)} \qquad \Longleftrightarrow \qquad \boxed{\int_{-\infty}^{t} \delta(\tau)\,d\tau = u(t)}$$

**证明思路**（用分部积分）：

$$\int_{-\infty}^{+\infty} \frac{du}{dt}\,\varphi(t)\,dt = \varphi(0)$$

这正好满足采样性质，所以 \(\frac{du}{dt}\) 就是 \(\delta(t)\)！

> 💡 **记忆口诀**：对阶跃求导得冲激，对冲激积分得阶跃。

---

#### 📝 奇异函数家族

| 函数 | 关系 |
|------|------|
| \(\cdots\) | \(\uparrow\) 求导 |
| \(\dot{\delta}(t)\)（冲激的导数） | ↑ |
| \(\delta(t)\)（单位冲激） | ↑ |
| \(u(t)\)（单位阶跃） | ↑ |
| \(r(t) = tu(t)\)（单位斜坡） | ↑ |
| \(\frac{t^2}{2}u(t)\)（单位抛物线） | ↑ |

## 📖 第4页：复指数函数 e^{st} 与复频率平面

### 1.4-3 指数函数 \(e^{st}\)

---

#### 🔵 复频率的引入

令复数 \(s = \sigma + j\omega\)，则：

$$\boxed{e^{st} = e^{(\sigma+j\omega)t} = e^{\sigma t}(\cos\omega t + j\sin\omega t)}$$

这里用到了**欧拉公式**：\(e^{j\omega t} = \cos\omega t + j\sin\omega t\)

| 参数 | 名称 | 作用 |
|------|------|------|
| \(\sigma\)（实部） | 奈培频率（neper frequency） | 控制幅度的增减速率 |
| \(\omega\)（虚部） | 弧度频率（radian frequency） | 控制振荡快慢 |

---

#### 🔵 e^{st} 涵盖的信号类型

\(e^{st}\) 是一个"超级通用"的信号，通过选择不同的 \(s\)，可以得到各种常见信号：

| 条件 | \(s\) 的值 | 信号类型 | 示例 |
|------|----------|----------|------|
| \(\sigma=0, \omega=0\) | \(s=0\) | 直流常数 | \(e^{0}=1\) |
| \(\omega=0\) | \(s=\sigma\)（实数） | 单调指数 | \(e^{\sigma t}\) |
| \(\sigma=0\) | \(s=\pm j\omega\) | 等幅正弦 | \(\cos\omega t\) |
| \(\sigma<0, \omega\neq0\) | \(s=\sigma\pm j\omega\)（左半平面） | 衰减正弦 | \(e^{-2t}\cos 5t\) |
| \(\sigma>0, \omega\neq0\) | \(s=\sigma\pm j\omega\)（右半平面） | 增幅正弦 | \(e^{2t}\cos 5t\) |

---

#### 🔵 实余弦的表示

由于 \(s^* = \sigma - j\omega\)，可以得到：

$$\boxed{e^{\sigma t}\cos\omega t = \frac{1}{2}(e^{st} + e^{s^*t})}$$

这说明**实值正弦信号**可以用**一对共轭复指数**来表示。

---

#### 🔵 复频率平面（s平面）⭐

\(s\) 平面是一个二维坐标系：
- **横轴**：实轴（\(\sigma\) 轴）
- **纵轴**：虚轴（\(j\omega\) 轴）

```
        jω
        ↑
        |          右半平面(RHP)
左半平面 |          增幅信号
(LHP)   |
衰减信号 |
————————+————————→ σ
        |
        |
        ↓
   虚轴上：等幅正弦
```

| 区域 | \(\sigma\) | 信号行为 |
|------|----------|----------|
| 左半平面（LHP） | \(\sigma < 0\) | 指数**衰减** |
| 虚轴 | \(\sigma = 0\) | **等幅**振荡 |
| 右半平面（RHP） | \(\sigma > 0\) | 指数**增长** |
| 实轴 | \(\omega = 0\) | 无振荡，单调变化 |
| 原点 | \(s = 0\) | 直流（常数） |

---

#### 🔵 具体例子

**例**：\(e^{-2t}\cos 5t\)（衰减正弦）

$$e^{-2t}\cos 5t = \frac{1}{2}\left[e^{(-2+j5)t} + e^{(-2-j5)t}\right]$$

复频率为 \(-2 \pm j5\)，位于**左半平面** ✓（衰减）

**例**：\(\cos 5t\)（等幅正弦）

$$\cos 5t = \frac{1}{2}\left[e^{j5t} + e^{-j5t}\right]$$

复频率为 \(\pm j5\)，位于**虚轴** ✓（等幅）

## 🎓 第1.4节 总结

---

### 📌 知识点一：单位阶跃函数 u(t)

$$u(t) = \begin{cases} 1 & t \geq 0 \\ 0 & t < 0 \end{cases}$$

**三大用途**：

| 用途 | 表达式 | 效果 |
|------|--------|------|
| 因果化 | \(f(t)\cdot u(t)\) | 信号从 \(t=0\) 开始 |
| 延迟门控 | \(u(t-a)-u(t-b)\) | 在 \([a,b)\) 上为1的窗口 |
| 分段表示 | 各段乘以对应门函数 | 用单一表达式描述分段信号 |

---

### 📌 知识点二：单位冲激函数 δ(t)

**Dirac定义**：
$$\delta(t)=0\ (t\neq 0), \quad \int_{-\infty}^{+\infty}\delta(t)\,dt=1$$

**乘法性质**：
$$\varphi(t)\,\delta(t-T) = \varphi(T)\,\delta(t-T)$$

**采样性质**（最重要！
