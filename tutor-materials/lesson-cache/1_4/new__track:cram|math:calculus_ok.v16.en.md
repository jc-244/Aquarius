## 📖 第2页：单位阶跃函数 $$u(t)$$

### 🎯 什么是单位阶跃函数？

想象一个电灯开关：**关着（值为0）→ 某一时刻突然打开（值变为1）**，这就是阶跃函数的直觉！

---

### 📐 数学定义

$$u(t) = \begin{cases} 1 & t \geq 0 \\ 0 & t < 0 \end{cases} \tag{1.8}$$

图形长这样：

```
u(t)
  1 |‾‾‾‾‾‾‾‾‾‾‾‾
    |
  0 |___
    ————————————→ t
        0
```

---

### 🔧 核心用途一：让信号"从零时刻开始"

如果我们有一个从 $$t = -\infty$$ 就存在的指数信号 $$e^{-at}$$，想让它**只在 $$t \geq 0$$ 时存在**，只需乘以 $$u(t)$$：

$$\text{因果信号} = e^{-at} u(t)$$

> 🎓 **记忆口诀**：乘以 $$u(t)$$ = 给信号装上"开关"，$$t<0$ 时全部清零！

---

### 🔧 核心用途二：描述矩形脉冲

一个在 $$t=2$$ 到 $$t=4$$ 之间值为1的矩形脉冲，可以写成：

$$x(t) = u(t-2) - u(t-4)$$

**理解方式**：
- $$u(t-2)$$：在 $$t=2$$ 时"打开"
- $$-u(t-4)$$：在 $$t=4$$ 时"关闭"

```
x(t)
  1 |     ‾‾‾‾‾‾‾‾
    |     |       |
  0 |_____         ______→ t
        2         4
```

---

### 🔧 核心用途三：描述分段函数（门控思想）

**例1.6 三角形信号的描述**

对于一个分段定义的三角形信号，可以拆成两部分：

$$x_1(t) = t\,[u(t) - u(t-2)]$$

$$x_2(t) = -2(t-3)\,[u(t-2) - u(t-3)]$$

最终合并：

$$\boxed{x(t) = t\,u(t) - 3(t-2)\,u(t-2) + 2(t-3)\,u(t-3)}$$

> 💡 **技巧**：$$[u(t-a) - u(t-b)]$$ 就是一个"门"，只在 $$a \leq t < b$$ 时值为1，其余为0。

---

### 📝 延迟阶跃函数

$$u(t - T)$$ 表示**延迟了 $$T$$ 秒**的阶跃函数：

$$u(t-T) = \begin{cases} 1 & t \geq T \\ 0 & t < T \end{cases}$$

## 📖 第3页：单位冲激函数 $$\delta(t)$$ 的定义与直觉

### 🎯 直觉理解：极限中的"瞬间爆发"

想象一个矩形脉冲，面积始终保持为1：
- 宽度越来越**窄** → 趋向于0
- 高度越来越**高** → 趋向于无穷大

当宽度趋向于0时，这个极限就是**单位冲激函数** $$\delta(t)$$！

```
高度 1/ε
    |  ‾‾|
    |    |        ε→0
    |    |   ————————→   δ(t) （无限高、无限窄、面积=1）
    |____|
   -ε/2  ε/2
```

---

### 📐 Dirac 的经典定义

$$\delta(t) = 0 \quad (t \neq 0) \qquad \text{且} \qquad \int_{-\infty}^{\infty} \delta(t)\,dt = 1 \tag{1.9}$$

> ⚠️ **注意**：$$\delta(t)$$ 在 $$t=0$$ 处**未定义**（不是无穷大，而是"未定义"），它不是普通意义上的函数！

---

### 🔢 冲激函数的强度

- **单位冲激** $$\delta(t)$$：面积 = 1
- **强度为 $$k$$ 的冲激** $$k\delta(t)$$：面积 = $$k$$

在图形中，用带括号的数字表示强度：

```
  ↑(k)
  |
——+——→ t
  0
```

---

### 🌟 其他近似冲激的函数

教材中提到，不只是矩形脉冲，以下函数在极限情况下也趋向冲激：

**指数脉冲**：$$\alpha e^{-\alpha t} u(t)$$，当 $$\alpha \to \infty$$ 时

验证：$$\int_0^{\infty} \alpha e^{-\alpha t}\,dt = 1$$ ✅（面积始终为1）

> 🎓 **关键思想**：冲激函数的本质不在于"形状"，而在于：
> 1. **有效宽度趋向于零**
> 2. **面积保持为1**

---

## 📖 第4页：冲激函数的重要性质

### 🌟 性质一：与函数相乘

设 $$\varphi(t)$$ 在 $$t=0$$ 处连续，则：

$$\boxed{\varphi(t)\,\delta(t) = \varphi(0)\,\delta(t)} \tag{1.10a}$$

**直觉**：冲激只在 $$t=0$$ 处"存活"，所以 $$\varphi(t)$$ 在其他地方的值都无关紧要，只有 $$\varphi(0)$$ 起作用。

**推广**（冲激在 $$t=T$$ 处）：

$$\boxed{\varphi(t)\,\delta(t-T) = \varphi(T)\,\delta(t-T)} \tag{1.10b}$$

---

### 🌟 性质二：采样性质（最重要！）

$$\boxed{\int_{-\infty}^{\infty} \varphi(t)\,\delta(t-T)\,dt = \varphi(T)} \tag{1.11}$$

**白话翻译**：用冲激 $$\delta(t-T)$$ 去"筛选"函数 $$\varphi(t)$$，结果就是 $$\varphi(t)$$ 在 $$t=T$$ 处的值！

> 🎓 **为什么叫"采样"？** 就像用针在 $$t=T$$ 处"戳"一下函数，取出那一点的值。这是数字信号处理的理论基础！

**推导过程**：

$$\int_{-\infty}^{\infty} \varphi(t)\,\delta(t-T)\,dt = \int_{-\infty}^{\infty} \varphi(T)\,\delta(t-T)\,dt = \varphi(T)\underbrace{\int_{-\infty}^{\infty}\delta(t-T)\,dt}_{=1} = \varphi(T)$$

---

### 🌟 性质三：冲激是阶跃的导数

$$\boxed{\frac{du(t)}{dt} = \delta(t)} \tag{1.12}$$

反过来：

$$\boxed{\int_{-\infty}^{t} \delta(\tau)\,d\tau = u(t)}$$

**直觉理解**：
- 阶跃函数在 $$t=0$$ 处"突然跳变"
- 跳变的"速率"无限大 → 就是冲激！

**图形理解**：

```
u(t) 的导数：
在 t=0 处斜率无穷大 → δ(t)
在其他地方斜率为0 → 0
```

---

### 📝 奇点函数族

从冲激函数出发，通过积分和微分可以得到一族函数：

$$\cdots \xrightarrow{\text{微分}} \dot{\delta}(t) \xrightarrow{\text{微分}} \delta(t) \xrightarrow{\text{积分}} u(t) \xrightarrow{\text{积分}} tu(t) \xrightarrow{\text{积分}} \frac{t^2}{2}u(t) \xrightarrow{\text{积分}} \cdots$$

这些统称为**奇点函数（Singularity Functions）**。

---

### 💡 广义函数的视角

严格来说，$$\delta(t)$$ 不是普通函数，而是**广义函数**：
- 不用"每点的值"来定义它
- 而用它对测试函数 $$\varphi(t)$$ 的**作用效果**（即采样性质）来定义它

---

## 📖 第5页：指数函数 $$e^{st}$$ 与复频率

### 🎯 为什么要引入复指数？

我们熟悉的函数——常数、指数衰减、正弦波、衰减正弦波——看起来各不相同，但其实都是同一个函数 $$e^{st}$$ 的特殊情况！

---

### 📐 复频率 $$s$$ 的定义

令 $$s = \sigma + j\omega$$（复数），则：

$$e^{st} = e^{(\sigma+j\omega)t} = e^{\sigma t} e^{j\omega t} = e^{\sigma t}(\cos\omega t + j\sin\omega t) \tag{1.13}$$

其中：
- $$\sigma$$：**奈培频率（实部）**，控制幅度的增减速率
- $$\omega$$：**角频率（虚部）**，控制振荡快慢

---

### 🗂️ 各种特殊情况一览

| 条件 | 信号类型 | 示例 |
|------|----------|------|
| $$s = 0$$ | 直流常数 | $$e^{0} = 1$$ |
| $$\omega = 0, s = \sigma$$ | 单调指数 | $$e^{\sigma t}$$ |
| $$\sigma = 0, s = \pm j\omega$$ | 等幅正弦 | $$\cos\omega t$$ |
| $$\sigma \neq 0, \omega \neq 0$$ | 指数调幅正弦 | $$e^{\sigma t}\cos\omega t$$ |

---

### 🗺️ 复频率平面（s平面）

```
        jω（虚轴）
         ↑
         |
LHP      |      RHP
（衰减） |    （增长）
         |
—————————+—————————→ σ（实轴）
         |
（虚轴上：等幅振荡）
```

| 位置 | 信号行为 |
|------|----------|
| **左半平面（LHP）**，$$\sigma < 0$$ | 指数**衰减**的振荡 |
| **虚轴**，$$\sigma = 0$$ | **等幅**振荡 |
| **右半平面（RHP）**，$$\sigma > 0$$ | 指数**增长**的振荡 |
| **实轴**，$$\omega = 0$$ | 单调增减（无振荡） |

---

### 🔢 实际例子

**例1**：$$e^{-2t}\cos 5t$$（衰减正弦）

$$= \frac{1}{2}\left(e^{(-2+j5)t} + e^{(-2-j5)t}\right)$$

复频率 $$s = -2 \pm j5$$ → 在**左半平面** ✅（衰减）

**例2**：$$e^{2t}\cos 5t$$（增长正弦）

复频率 $$s = 2 \pm j5$$ → 在**右半平面** ✅（增长）

**例3**：$$\cos 5t$$（等幅正弦）

复频率 $$s = \pm j5$$ → 在**虚轴上** ✅（等幅）

---

### 💡 重要公式

由 Euler 公式可得：

$$e^{\sigma t}\cos\omega t = \frac{1}{2}\left(e^{st} + e^{s^*t}\right) \tag{1.14}$$

其中 $$s^* = \sigma - j\omega$$ 是 $$s$$ 的共轭。
